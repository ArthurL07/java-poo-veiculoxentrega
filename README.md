🚚 Sistema de Controle de Veículos e Entregas 
-----------------------------------------------------------------------------------------------------------------
Este projeto foi desenvolvido a partir de uma regra de negócio criada por mim, simulando um pequeno sistema de gerenciamento de veículos e entregas.
A proposta foi aplicar os conceitos de Programação Orientada a Objetos (POO) em Java, utilizando apenas os recursos solicitados pelo professor, como construtores, métodos, herança e encapsulamento.

📖 Contexto do Projeto
----------------------------------------------------------------------------------------------------------------
Empresas de logística e transporte precisam gerenciar sua frota de veículos e entregas de forma organizada para garantir eficiência nas operações.

Pensando nisso, foi desenvolvido um sistema simples que permite:

Cadastrar veículos e entregas
Associar entregas aos veículos responsáveis
Controlar informações de forma estruturada

⚙️ Funcionalidades
-----------------------------------------------------------------------------------------------------------------
Armazena informações sobre veículos, como:
modelo
capacidade de carga
quantidade de eixos
se possui baú
Armazena informações sobre motocicletas, como:
modelo
capacidade de carga
cilindradas
se possui baú
Permite registrar entregas com descrição
Permite associar entregas a veículos
Exibe os dados completos dos veículos e entregas
Utiliza conceitos de POO com:
classes
herança
atributos privados
métodos públicos

💻 Tecnologias e Conceitos Utilizados
------------------------------------------------------------------------------------------------------------------
Linguagem: Java
Conceitos de POO: Classes, Objetos, Construtores, Herança, Encapsulamento
Execução: Via console (linha de comando)
